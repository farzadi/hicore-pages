Hicore

Hicore is a free and open-source game server.

Get Started →

Easy To Install

Minimal setup with Docker helps you focus on your game development.

Fast-paced Games

It's developed to handle fast-paced games. We use the RUDP protocol, so you can develop any online and real-time games without worrying about network delays.

Dashboard

The dashboard feature allows you to control your game like user progress, match rules, create collection data for your game, analyze your game, network, users, and more.

Social

Players can add their friends and create rooms to play together or play against opponents.

Matchmaking

With the matchmaking system, the server puts opponents against each other based on their rank, level, and other parameters.

Client SDK

We support the Unity3D Engine. Other platforms, such as Unreal Engine, will be added soon.
